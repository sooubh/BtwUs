Built on Local-First Encryption

Hardware-level protection designed to shield intimate moments.

AES-256 Encryption

All stored textual records, commitments, and tags are compiled in a local SQLite file which is directly encrypted via SQLCipher using a key derived via PBKDF2 algorithm.

Biometric Authorization

Secure the application with Android local biometric APIs (Fingerprint or Face Unlock). Your biometrics check is handled securely by the OS kernel, keeping it invisible to the application.

Decoy Vault (Plus/Premium)

Configure a fake secondary passcode that redirects the application to a blank, normal decoy vault. Ensures complete peace of mind if you are pressured to unlock your application.
